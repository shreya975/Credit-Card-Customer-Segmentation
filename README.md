An AI-powered customer segmentation project that groups credit card customers based on their spending behavior, purchasing habits, and financial activities using Machine Learning. The project helps financial institutions identify valuable customer segments for targeted marketing and personalized financial services.

---

# 📌 Project Overview

Understanding customer behavior is essential for banks and financial institutions to improve customer satisfaction, optimize marketing strategies, and increase profitability.

This project applies **Unsupervised Machine Learning** to cluster customers with similar spending patterns and financial behaviors. The resulting customer segments enable businesses to design personalized offers, loyalty programs, and risk management strategies.

# 🤖 Machine Learning Model

The project uses

## K-Means Clustering

to group customers into multiple clusters based on their financial behavior.

The optimal number of clusters is determined using the **Elbow Method**.
